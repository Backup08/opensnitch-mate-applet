Mate Status Applet for Open Snitch Firewall.
Shows if <a href="https://github.com/evilsocket/opensnitch">Opensnitch 
Firewall</a> is running.


<h2>Install</h2>
Run sudo ./make.sh to install
Then add to panel in MATE Desktop
